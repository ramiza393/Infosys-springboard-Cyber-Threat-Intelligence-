Cybersecurity Threat Intelligence Project

**Overview**

This project implements a threat scoring system to prioritize cybersecurity threats based on their severity and impact.

**Features**

- Assigns numeric scores to attack techniques based on keywords and descriptions.
- Filters critical threats based on a user-defined threshold.
- Enables analysts to focus on high-risk threats and optimize resource allocation.

**Usage**

1. Input a list of attack techniques with descriptions.
2. Define keyword weights and scoring rules.
3. Run the scoring system to generate a ranked list of threats.

**Example Output**
Technique	Score
Tech1	8.9
Tech2	7.5
Tech3	5.0
